Ir a https://platform.openai.com/api-keys - Crear nueva clave 2. **Instalar cliente** ```bash npm install openai ``` 3. **Usar en código** ```javascript import { OpenAI } from 'openai'; const openai = new OpenAI({ apiKey: process.env.OPENAI_API_KEY }); async function generateContent(prompt) { const message = await openai.chat.completions.create({ model: 'gpt-4', messages: [{ role: 'user', content: prompt }] }); return message.choices[0].message.content; } ``` ### Hostinger Deployment 1. **Preparar archivos** ```bash # Asegúrate de que tienes: - index.html - style.css - app.js - README.md ``` 2. **Subir via FTP** - Conectar a Hostinger via FTP - Subir archivos a carpeta `public_html` 3. **Dominio personalizado** - Apunta tu dominio a Hostinger -
